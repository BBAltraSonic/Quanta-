This folder holds SVG icon assets used throughout the UI.

Expected files (placeholders can be replaced with your real designs):
- search.svg
- volume.svg
- menu.svg
- like.svg
- comment.svg
- share.svg
- save.svg
- home.svg
- profile.svg
- plus.svg (optional if not using Material 'add' icon)

Icon requirements:
- ViewBox: 24x24
- No hardcoded fills; use fill="currentColor" so we can color via Flutter.
- No strokes that ignore color unless intended.

Temporary developer placeholders can be used until final assets are dropped in.
